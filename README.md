## 🧾 Description
This project is a comprehensive **Investment Banking Database Management System** built using SQL Server. It simulates real-world financial operations such as client management, securities trading, portfolio tracking, and market analysis.

The project also includes advanced SQL queries, views, and analytical solutions to generate meaningful financial insights like portfolio performance, risk analysis, and transaction summaries.

---

## 🚀 Features
- Client management with risk profiling  
- Securities management (Stocks, Bonds, ETFs, etc.)  
- Transaction tracking (Buy/Sell operations)  
- Portfolio performance monitoring  
- Research reports with analyst recommendations  
- Market data tracking (daily price movements)  
- Advanced SQL analytics and reporting  

---

## 🛠 Tech Stack
- **Database:** SQL Server  
- **Language:** T-SQL  
- **Tools:** SQL Server Management Studio (SSMS)  

---

## 📂 Database Schema

### Tables:
- **Clients** – Stores client details and risk profiles  
- **Securities** – Contains financial instruments data  
- **Transactions** – Records buy/sell activities  
- **Portfolio** – Tracks client holdings  
- **ResearchReports** – Analyst insights and recommendations  
- **MarketData** – Historical price data  

---

## 📈 Key SQL Implementations
- Views for portfolio summary and top-performing securities  
- Complex joins for transaction and client insights  
- Window functions (RANK, DENSE_RANK, AVG)  
- Common Table Expressions (CTEs)  
- Stored procedures for transaction handling  
- Aggregations for financial metrics  

---

## 🔍 Sample Queries & Insights

### 1. Client Portfolio Summary
- Total portfolio value  
- Number of holdings

### 2. Transaction Analysis

### 3. Market Performance

- Top-performing securities  

### 4. Risk Analysis

### 5. Profit/Loss Calculation
- Unrealized gain/loss per portfolio

## ⚙ Installation

1. Install **SQL Server** and **SSMS**
2. Open SQL Server Management Studio
3. Copy and run the SQL script provided in this repository
4. The database `InvestmentBanking` and all tables will be created automatically
